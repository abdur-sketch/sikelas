# SIKELAS Nurul Iman

MVP web responsif untuk Sistem Informasi Kelas dan Pembelajaran SMK Nurul Iman.

## Fitur demo

- login guru
- dashboard ringkasan kelas
- pencarian dan tabel data siswa
- pencatatan absensi interaktif
- jadwal pelajaran mingguan
- pemantauan tugas dan rekap nilai

## Menjalankan aplikasi

Prasyarat: Node.js 22.13 atau lebih baru.

```bash
npm install
npm run dev
```

Buka alamat lokal yang ditampilkan pada terminal. Data pada versi ini adalah data contoh dan belum tersambung ke basis data produksi.

## Tahap berikutnya

Hubungkan data ke basis data, tambahkan autentikasi berbasis peran, serta aktifkan modul materi, pengumuman, dan portofolio siswa.
